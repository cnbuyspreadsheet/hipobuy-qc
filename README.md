## QC workflow

| Counter | What it proves | Buyer action |
|---|---|---|
| Identity | Item, size, color, label, accessory count | Match photos to the linked product page |
| Construction | Shape, stitching, glue, print, embroidery | Retake only if a new angle changes the verdict |
| Measurement | Insole, chest, length, waist, sleeve | Trust tape photos over vague listing titles |
| Release | Packaging, missing photos, final risk | Ship, retake, replace, or cancel |

## Signal board

| Signal | Outcome |
|---|---|
| Blurry label or hidden tag | Ask for one close-up retake |
| Size-sensitive item | Request tape-measure proof |
| Same defect from multiple angles | Replace or cancel if it matters |
| Clean identity + normal minor marks | Release the parcel |

## Guide counters

| Guide | Purpose |
|---|---|
| Photo Evidence Counter | Angles, labels, lighting, warehouse proof |
| Size Tag Audit | Tags, insole shots, garment measurements |
| Stitch and Shape Review | Seams, panels, glue, embroidery, print drift |

| Retake Ticket Script | Short copy for better QC photo requests |
| Parcel Release Gate | Final pre-shipping checklist |
| Shoe QC Lane | Sole, upper, heel, box, pair matching |
| Apparel QC Lane | Hoodies, jerseys, tees, jackets, pants, sets |

## Retake snippet

```text
Please retake a clear photo of [area] in bright light.
Please place a tape measure on [length/insole/chest/waist].
Please show both sides of [shoe/label/accessory] in one photo.
```

Independent guide. Product listings, prices, QC options, agent availability, and shipping details may change. Always verify details on the linked product page and selected shopping agent before ordering.
